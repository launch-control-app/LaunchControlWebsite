# Launch Control Website

## Server

To boot up the server go into the server directory and run

```
yarn install
```
then run
```
yarn start
```

## Frontend

#### Development Mode

To boot up the frontend go into the frontend directory and run

```
yarn install
```
then run
```
yarn start
```
